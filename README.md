# movie_app

A feature-rich movie application built with Flutter that enables users to discover, search, and
explore a wide range of movies. The app integrates with an external API to fetch real-time movie
data, including ratings, genres, and detailed descriptions. It follows clean architecture principles
and provides a responsive and smooth user experience.
